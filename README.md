# Bajil Smart Village - GIS & Scientific Intelligence Suite
نظام التحليل الجيومكاني المتطور ومحاكاة السيول - مشروع القرية النموذجية بباجل

## 🚀 نظرة عامة
هذا المشروع هو منصة هندسية متكاملة تستخدم تقنيات هندسة الجيوماتكس، الذكاء الاصطناعي (MCDA)، والنمذجة الهيدرولوجية لتحليل تضاريس منطقة باجل. تهدف المنصة إلى توفير دعم قرار ذكي لاختيار مواقع الكرفانات وتأمينها من مخاطر الطبيعة.

## 📊 مسار العمل المتكامل (The Pipeline)
1.  **DEM Engine**: توليد نموذج الارتفاع الرقمي بدقة عالية.
2.  **Hydro-Conditioning**: تصحيح التضاريس وإزالة العيوب الطبوغرافية.
3.  **Flow Analysis**: تحديد مسارات الجريان السطحي والعمود الفقري المائي للموقع.
4.  **Scientific MCDA**: نموذج قرار يعتمد على المنطق الضبابي لاختيار أفضل المناطق الإنشائية.
5.  **DBSCAN Clustering**: تجميع بكسلات الملاءمة في مناطق (Zones) عمرانية متماسكة.
6.  **Flood Simulation (جديد)**: محاكاة ديناميكية لسقوط الأمطار وتحديد أعماق الغمر ومناطق الخطر.

## 🌧️ محاكاة الفيضانات (Phase 6)
تمت إضافة محاكي هيدروليكي يقوم بـ:
- محاكاة سقوط **50 ملم** من الأمطار.
- تتبع حركة المياه عبر **15 دورة تكرارية**.
- عرض النتائج في نموذج **3D تفاعلي** يوضح طبقة الفيضانات باللون الأزرق المتوهج فوق التضاريس.

## 🏆 المخرجات والتقارير (Deliverables)
### البيانات المكانية
- `suitability_zones.shp`: المضلعات الهندسية لأفضل 70 منطقة بناء.
- `flood_zones.shp`: خارطة مناطق خطر الفيضانات المستخرجة آلياً.
- `flood_map.tif`: راستر يوضح أعماق تجمع المياه في كامل الموقع.

### التقارير البصرية التفاعلية
- **3D Flood Model**: `interactive_water_map.html` - نموذج تفاعلي لمسارات السيول.
- **Suitability Dashboard**: `interactive_3d_suitability_model.html` - لوحة التحكم في اختيار المواقع.
